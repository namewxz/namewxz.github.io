This is a collection of over fifty logic puzzles and games you can play right in your web browser. Each puzzle has multiple difficulty levels and variations, with unlimited, randomly generated games. And it’s all completely free: no upsells, no ads, no trackers.

This web app should run in any recent browser. You can use a touch screen, mouse or keyboard. It even works offline.

Enjoy!

P.S., Looking for free Sudoku? It’s called Solo here, also available in diagonal (X-style), jigsaw, killer, and mega (Hexadoku) variants. Mathdoku is called Keen. And if you like that type of puzzle, also check out Towers and Unequal.

⚠️ This app is still actively being developed. While it should be (mostly) usable, there are 🚧 incomplete parts, 🥽 experiments in progress, and 🪲🐛🪰 plenty of bugs. Things might change or break without warning.
