第 4 章：立方体

这是我最初作为网页游戏看到的另一个游戏。这个是 Paul Scott 的 Java 游戏 [2]。你有一个 16 个方块的网格，其中六个是蓝色的；一个方块上放着一个立方体。你的移动是使用方向键将立方体滚动 90 度，使其移动到相邻的方块。如果你将立方体滚到蓝色方块上，蓝色方块会被立方体的一个面上；如果你将立方体的蓝色面滚到非蓝色方块上，蓝色会再次放下。（一般来说，每当你滚动立方体时，接触的两个面会交换颜色。）你的任务是同时将所有六个蓝色方块放到立方体的六个面上。计算你的移动次数，并尝试用尽可能少的移动次数完成。

与原始的 Java 游戏不同，我的版本有一个额外的功能：一旦你掌握了在正方形网格上滚动立方体的游戏，你可以切换到三角形网格并滚动四面体、八面体或二十面体中的任何一个。

[2] http://www3.sympatico.ca/paulscott/cube/cube.htm

4.1 立方体控制

这个游戏可以用键盘或鼠标玩。

左键点击窗口上的任何地方都会将立方体（或其他固体）移向鼠标指针。

方向键也可以用于在正方形网格上向四个基本方向滚动立方体。在三角形网格上，方向键到方向的映射更近似。在没有意义的地方不允许垂直移动。数字键盘上方向键周围的四个键（'7'，'9'，'1'，'3'）可用于对角线移动。

（第 2.1 节 中描述的所有操作也可用。）

4.2 立方体参数

这些参数可从“类型”菜单的“自定义...”选项获得。

固体类型
选择要滚动的固体（并因此选择网格的形状）：四面体、立方体、八面体或二十面体。
宽度 / 顶部，高度 / 底部
在正方形网格上，水平和垂直尺寸。在三角形网格上，顶行和底行的三角形数量。
