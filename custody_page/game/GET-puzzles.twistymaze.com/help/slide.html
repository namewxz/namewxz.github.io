Slide

Slide the blocks to move the blue key block to the green exit area. Blocks can slide up, down, left or right into empty spaces. Only the key block can slide into the exit gate.

Use the mouse to drag the blocks around.

Status: This is an experimental, unfinished puzzle. It mostly works, but may have bugs or unexpected behavior. Random game generation may be slow, particularly at larger sizes. (Be patient.) Keyboard control is not yet supported.
