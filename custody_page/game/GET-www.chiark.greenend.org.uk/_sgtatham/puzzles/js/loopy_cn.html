    如果你已经阅读此消息超过一两秒钟，那么 这个 WebAssembly 拼图似乎无法在你的网络浏览器中工作。抱歉！

    更多信息

    大多数现代浏览器应该支持 WASM。我收到了以下浏览器的成功报告：

    • Firefox 87.0
    • Chrome 89.0.4389.114
    • Safari 14.0.3 (16610.4.3.1.7)
    • Edge 89.0.774.68
    • Opera 75.0.3969.149

    如果这个拼图在这些浏览器之一（或更新版本）中不工作，请检查你是否有任何本地配置可能完全关闭了 WebAssembly，或者某种阻止软件可能阻止了 WASM 代码首先被下载。

    （例如，在 Firefox 中，可以通过在 about:config 界面中将 javascript.options.wasm 设置为 false 来完全关闭 WASM。如果你在 Firefox 或其他浏览器中做了类似的操作，这个拼图将无法运行。）

    在其他浏览器中，问题可能是 WebAssembly 根本不被支持（例如，Internet Explorer 11），或者需要浏览器更新。

    如果你认为你的浏览器 应该 支持 WebAssembly，但这个拼图仍然无法运行，请报告问题，包括 尽可能多的诊断信息。

    特别是，尝试打开浏览器的 Javascript 错误控制台，然后重新加载此页面，并告诉我它是否报告任何错误消息。

    此外，如果你的浏览器有网络诊断选项卡，请尝试相同的实验，确保它成功加载了两个辅助文件 loopy.js 和 loopy.wasm。
