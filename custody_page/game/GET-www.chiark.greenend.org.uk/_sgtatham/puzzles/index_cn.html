Simon Tatham的便携益智游戏集合

介绍

本页面包含一系列小型计算机程序，实现了单人益智游戏。所有游戏都可以在Unix（GTK）和Windows上原生运行。它们也可以在网络上作为Java或Javascript小程序游玩。

2023-09-20：很抱歉，我将不再能够构建这些游戏的MacOS版本。我的Mac已经很旧了，一直在挣扎运行；现在它已经停止工作，我也不打算购买新的。如果有人愿意自愿维护Mac前端代码并提供可下载的应用程序，我会帮助他们开始，并在网站启动后链接到他们的站点。对不起，Mac用户们。与此同时，总有网络版本可用...

我编写这个集合是因为我认为应该有更多的小型桌面玩具：你可以在窗口中弹出并玩上两三分钟的小游戏，当你从其他事情中休息时。我也很恼火，每次我在（比如说）Unix上找到一个好游戏，下次我坐在Windows机器前时它就不可用了，反之亦然；所以我安排了我的个人益智游戏集合中的所有游戏都能在这两个平台上愉快地运行，甚至更多平台。当我发现（或者可能发明）更多我喜欢的益智游戏时，它们会被添加到这个集合中，并立即在两个平台上可用。如果有人想为我不支持的平台编写其他前端（已经有人这样做了），那么这个框架中的所有游戏将立即在另一个平台上可用。

游戏

这个集合中的实际游戏大多不是我的发明；它们是在我的便携益智游戏框架内对现有游戏概念的重新实现。一般来说，我不声称拥有这些谜题规则的发明权。（我甚至不声称拥有所有代码的作者权；下面的一些谜题是由其他作者提交的。）

每张图片下方都有两个链接，指向你可以在网络上玩的谜题版本。'js'链接（以及谜题图片本身）链接到基于JavaScript和WebAssembly的版本。'java'链接现在可能已经过时：它链接到Java小程序，大多数浏览器不再支持。

每张图片下方还有一个指向游戏Windows二进制文件的链接，以及一个指向手册的链接。对于所有其他平台，游戏以单个包的形式提供，因此请向下滚动到下载部分获取所有游戏。

开发

所有这些拼图都是用C语言编写的，带有移植接口，以便相同的后端拼图代码可以与完全不同的图形前端通信。上面的源归档包括Windows和Mac OS X的原生GUI前端，使用GTK+库的X前端，以及用于使用NestedVM将拼图编译为Java小程序的混合C/Java前端。

有广泛的开发者文档描述跨平台接口。如果你想编写新的拼图或新的前端（使所有这些游戏在另一个平台上运行），这可能是开始的地方。

如果你想查看开发的最新状态，你可以从我的git存储库中检出开发源代码： 

git clone https://git.tartarus.org/simon/puzzles.git

或者，你可以在网上浏览存储库，这里。

反馈

请向anakin@pobox.com报告错误。

如果你的错误报告涉及特定拼图有问题（如果你认为它没有解决方案，或有多个解决方案），请尽可能发送保存的游戏文件，使用'游戏>保存'菜单选项或等效选项。保存的游戏文件比图形屏幕截图更有用（且更小）：它们包含我需要重现游戏生成并找出问题所在的所有信息。

如果你不能这样做（例如，如果你在网络上玩，它没有保存游戏功能），请至少复制并粘贴随机种子和游戏描述（从'随机种子'和'特定'菜单选项，或Javascript游戏中的'输入游戏ID'和'输入随机种子'按钮）。

此外，你可能会发现在报告错误之前阅读这篇文章会有所帮助。
