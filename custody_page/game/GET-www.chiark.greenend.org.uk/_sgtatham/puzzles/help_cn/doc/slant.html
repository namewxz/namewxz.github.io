第 20 章：斜线

你有一个方格网格。你的目标是在每个方格中画一条对角线，并选择每条线的倾斜方向，以便满足以下条件：

  • 对角线永远不会形成一个环。
  • 任何带有圈出数字的点恰好有那么多线在那里交汇。（因此，4 是十字形的中心，而 0 是菱形的中心 —— 或者更确切地说，是部分菱形的中心，因为 0 永远不能出现在网格的中间，因为那会立即导致一个环。）

这个拼图的功劳归于 Nikoli [8]。

[8] https://www.nikoli.co.jp/ja/puzzles/gokigen_naname/（日语）

20.1 斜线控制

左键点击空白方格将在其中放置一个 \（向左倾斜的线，即从方格的左上角延伸到右下角）。右键点击空白方格将在其中放置一个 /（向右倾斜，从右上角延伸到左下角）。

继续点击任一按钮将在三种可能的方格内容之间循环。因此，如果你在空白方格中反复左键点击，它将从空白变为 \ 变为 / 变回空白，如果你反复右键点击，方格将从空白变为 / 变为 \ 变回空白。（因此，如果你需要，你可以完全用一个按钮玩游戏。）

你也可以使用光标键在网格周围移动。按回车键或空格键将分别放置 \ 或 /，然后如上所述循环它们。你也可以按 / 或 \ 分别放置 / 或 \，与光标方格中已有的内容无关。退格键从光标方格中移除任何线。

（第 2.1 节 中描述的所有操作也可用。）

20.2 斜线参数

这些参数可从“类型”菜单的“自定义...”选项获得。

宽度，高度
网格的大小（以方块为单位）。
难度
控制生成的拼图的难度。在困难级别，你需要基于方格之间的关系知识进行推理，而不是总是能够一次推断出一个方格的确切内容。（例如，你可能知道两个方格向同一方向倾斜，即使你还不知道那个方向是什么，这可能使你能够推断出关于其他方格的信息。）即使在困难级别，也永远不需要猜测和回溯。

20.3 斜线用户首选项

在支持用户首选项的平台上，“游戏”菜单上的“首选项”选项将允许你配置鼠标按钮的工作方式。
